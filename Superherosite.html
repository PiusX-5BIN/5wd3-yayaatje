Project: Superhero Website

Voor mijn allereerste project moest ik een website bouwen over een bepaalde superheld. Omdat ik zelf een grote Marvel-fan ben heb ik gekozen voor Spider-man. Deze website bestaan o.a. uit een Profile, waarin info over de superheld wordt gegeven. Er staan ook getuigen bij van mensen die Spider-man kennen. En ten slotte staan er ook enkele Spiderman-comics bij met de naam en de release date. Het bouwen van deze website was voor mij een hele uitdaging. Het ging nogal moeizaam, ik heb veel geduld moeten hebben. Er zijn enkele dingen aan deze website waar ik echt mee vast zat. Sommige dingen wist ik uiteindelijk uit te vogelen, andere niet. Ik ben er dan ook enkel in geslaagd om een mobile versie van mijn website te maken.
